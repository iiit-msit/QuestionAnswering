codes</h1>\n\t<div id="bodyContent" class="mw-body-content">\n\t\t<div id="siteSub" class="noprint">From Wikipedia, the free encyclopedia</div>\n\t\t<div id="contentSub"></div>\n\t\t<div id="contentSub2"></div>\n\t\t\n\t\t<div id="jump-to-nav"></div>\n\t\t<a class="mw-jump-link" href="#mw-head">Jump to navigation</a>\n\t\t<a class="mw-jump-link" href="#searchInput">Jump to search</a>\n\t\t<div id="mw-content-text" lang="en" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><div class="shortdescription nomobile noexcerpt noprint searchaux" style="display:none">Wikimedia list article</div>\n<p>Unique, fixed <b>customer codes</b> were used by <a href="/wiki/Boeing_Commercial_Airplanes" title="Boeing Commercial Airplanes">Boeing Commercial Airplanes</a> to denote the original customer for airframes produced as part of Boeing\'s 7x7 family of commercial aircraft from 1956 (except for the <a href="/wiki/Boeing_787_Dreamliner" title="Boeing 787 Dreamliner">Boeing 787 Dreamliner</a>) , with the introduction of the <a href="/wiki/Boeing_707" title="Boeing 707">707</a>, up till 2016.\n</p><p>The codes are in the form of two letters and/or numbers which are appended to the aircraft\'s model designator, as seen in the following examples:\n</p>\n<ul><li>The <a href="/wiki/Boeing_727" title="Boeing 727">Boeings 727-100 and 727-200</a> ordered by <a href="/wiki/United_Airlines" title="United Airlines">United Airlines</a> with customer code 22 are designated as 727-22 and 727-222 respectively.</li>\n<li>A <a href="/wiki/Boeing_737_Next_Generation" title="Boeing 737 Next Generation">Boeing 737-800</a> ordered by <a href="/wiki/Southwest_Airlines" title="Southwest Airlines">Southwest Airlines</a> with customer code H4 would be designated as Boeing 737-8H4.</li>\n<li>A <a href="/wiki/Boeing_747-400" title="Boeing 747-400">Boeing 747-400</a> ordered by <a href="/wiki/Korean_Air" title="Korean Air">Korean Air</a> with customer code B5 would be designated as Boeing 747-4B5.</li>\n<li>A <a href="/wiki/Boeing_747SP" title="Boeing 747SP">Boeing 747SP</a> ordered by <a href="/wiki/Qantas" title="Qantas">Qantas</a> with customer code 38 would be designated as Boeing 747SP-38.</li>\n<li>A <a href="/wiki/Boeing_747" title="Boeing 747">Boeing 747</a>SR ordered by <a href="/wiki/Japan_Airlines" title="Japan Airlines">Japan Airlines</a> with customer code 46 would be designated as Boeing 747SR-46, rather than 747-146(SR).</li>\n<li>A <a href="/wiki/Boeing_767" title="Boeing 767">Boeing 767-300ER</a> ordered by <a href="/wiki/EVA_Air" title="EVA Air">EVA Air</a> with customer code 5E would be designated as Boeing 767-35E(ER).</li>\n<li>A <a href="/wiki/Boeing_777" title="Boeing 777">Boeing 777F</a> ordered by <a href="/wiki/GECAS" class="mw-redirect" title="GECAS">GECAS</a> with customer code 6N would be designated as Boeing 777-F6N.</li>\n<li>A <a href="/wiki/Boeing_777" title="Boeing 777">Boeing 777-300ER</a> ordered by <a href="/wiki/Cathay_Pacific" title="Cathay Pacific">Cathay Pacific</a> with customer code 67 would be designated as Boeing 777-367(ER).</li></ul>\n<p>The codes do not change if the aircraft is subsequently sold, such as if a passenger aircraft is converted into a freighter or other purposes. For example, a <a href="/wiki/Boeing_Dreamlifter" title="Boeing Dreamlifter">Boeing Dreamlifter</a> that was originally built as a 747-400 passenger aircraft for <a href="/wiki/China_Airlines" title="China Airlines">China Airlines</a> with customer code 09 would be designated as 747-409(LCF). Although in some cases if an airline cancelled or sold their order before Boeing had delivered the airframe the customer code would be changed to that of the new purchaser. One such example is the order for sixteen 737-800s taken over by <a href="/wiki/Qantas" title="Qantas">Qantas</a> from <a href="/wiki/American_Airlines" title="American Airlines">American Airlines</a> after the <a href="/wiki/September_11_attacks" title="September 11 attacks">September 11 attacks</a> in 2001 - these aircraft were delivered with Qantas\' 38 code rather than 23 for American.\n</p><p>Before the 707, Boeing used a generally similar system to identify the presence of detailed variations or options requested by particular customers, but the codes were not customer-specific.\n</p><p>In 2016, Boeing announced that they would no longer apply customer codes to any aircraft produced after a certain point, which would lead to their designators being the "generic" type for the model. The codes were removed from the type certificates for each model with effect from the production line number shown below:\n</p>\n<ul><li><a href="/wiki/Boeing_737_Next_Generation" title="Boeing 737 Next Generation">Boeing 737 Next Generation</a>: line number 6082</li>\n<li><a href="/wiki/Boeing_P-8_Poseidon" title="Boeing P-8 Poseidon">Boeing P-8 Poseidon</a>: line number 6020</li>\n<li><a href="/wiki/Boeing_747-8" title="Boeing 747-8">Boeing 747-8</a>: line number 1534</li>\n<li><a href="/wiki/Boeing_767" title="Boeing 767">Boeing 767</a>: line number 1102</li>\n<li><a href="/wiki/Boeing_777" title="Boeing 777">Boeing 777</a>: line number 1422</li></ul>\n<p>Further, customer codes have never been used for the <a href="/wiki/Boeing_787_Dreamliner" title="Boeing 787 Dreamliner">787</a> and <a href="/wiki/Boeing_737_MAX" title="Boeing 737 MAX">737 MAX</a>, and will not be applied to the <a href="/wiki/Boeing_777X" title="Boeing 777X">777X</a> either.\n</p>\n<div
