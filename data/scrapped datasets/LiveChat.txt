customer service software with <a href="/wiki/Online_chat" title="Online chat">online chat</a>, <a href="/wiki/Help_desk" title="Help desk">help desk software</a>, and <a href="/wiki/Web_analytics" title="Web analytics">web analytics</a> capabilities. \n</p><p>It was first launched in 2002<sup id="cite_ref-launch_2-0" class="reference"><a href="#cite_note-launch-2">&#91;2&#93;</a></sup> and is currently developed and offered in a SaaS (<a href="/wiki/Software_as_a_service" title="Software as a service">software as a service</a>) business model by <a href="/wiki/Livechat_Software" title="Livechat Software">LiveChat Software S.A.</a>.\n</p><p>Companies use LiveChat as a <a href="/wiki/Service_desk_(ITSM)" class="mw-redirect" title="Service desk (ITSM)">single point of contact</a> to manage from one software all <a href="/wiki/Customer_service" title="Customer service">customer service</a> and <a href="/wiki/E-commerce" title="E-commerce">online sales</a> activities that normally are provided using different channels (chat, <a href="/wiki/Email" title="Email">email</a> and <a href="/wiki/Social_media" title="Social media">social media</a>) and multiple tools.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup> LiveChat is used by over 30,000<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup> paid customers from over 150 countries.<sup id="cite_ref-technadu_5-0" class="reference"><a href="#cite_note-technadu-5">&#91;5&#93;</a></sup><sup id="cite_ref-6" class="reference"><a class="mw-editsection-bracket">]</span></span></h2>\n<p>Technically, LiveChat is a <a href="/wiki/Proprietary_software" title="Proprietary software">proprietary software</a> built with the use of multiple technologies, including <a href="/wiki/JavaScript" title="JavaScript">JavaScript</a>, <a href="/wiki/Node.js" title="Node.js">Node.js</a>, <a href="/wiki/MySQL" title="MySQL">MySQL</a>, <a href="/wiki/ElasticSearch" class="mw-redirect" title="ElasticSearch">ElasticSearch</a>, <a href="/wiki/C%2B%2B" title="C++">C++</a> and <a href="/wiki/Erlang_(programming_language)" title="Erlang (programming language)">Erlang</a>.<sup id="cite_ref-7" class="reference"><a href="#cite_note-7">&#91;7&#93;</a></sup> The entire platform consists of two main elements: \n</p>\n<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/File:Chats_in_LiveChat_application.png" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/en/thumb/a/a3/Chats_in_LiveChat_application.png/220px-Chats_in_LiveChat_application.png" decoding="async" width="220" height="137" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/en/thumb/a/a3/Chats_in_LiveChat_application.png/330px-Chats_in_LiveChat_application.png 1.5x, //upload.wikimedia.org/wikipedia/en/a/a3/Chats_in_LiveChat_application.png 2x" data-file-width="399" data-file-height="249" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/File:Chats_in_LiveChat_application.png" class="internal" title="Enlarge"></a></div>LiveChat application viewed as agent</div></div></div>\n<ul><li><b>Website chat widget</b> - embedded on customer\'s website and seen by the end users (website visitors) as a small chat box, traditionally displayed in the bottom right corner of the <a href="/wiki/Web_browser" title="Web browser">web browser</a>. Chat widget is visible on desktop and mobile devices, either in the browser or as a part of mobile application.</li>\n<li><b>Agent application</b> - used by the company employees to respond to questions asked by the customers. Usually agents use the <a href="/wiki/Web_application" title="Web application">web-based</a> application, which allows employees to log in and conduct a chat with website visitor using any browser. However, there are desktop applications available for <a href="/wiki/Windows" class="mw-redirect" title="Windows">Windows</a> and <a href="/wiki/Mac_OS_X" class="mw-redirect" title="Mac OS X">Mac OS X</a> and mobile apps for <a href="/wiki/Android_(operating_system)" title="Android (operating system)">Android</a> and <a href="/wiki/IOS_(Apple)" class="mw-redirect" title="IOS (Apple)">iOS</a>. <a href="/wiki/BlackBerry_OS" title="BlackBerry OS">BlackBerry</a> and <a href="/wiki/Windows_Phone" title="Windows Phone">Windows Phone</a> mobile apps used to exist in the past, but are no longer developed. Agent application is enhanced with <a href="/wiki/User_interface_techniques" class="mw-redirect" title="User interface techniques">user interface tools</a> that make <a href="/wiki/Point-to-multipoint_communication" title="Point-to-multipoint communication">one-to-many</a> communication quicker and more efficient. Examples of such tools include: <a href="/wiki/Canned_response" title="Canned response">canned responses</a>, <a href="/wiki/Keyboard_shortcut" title="Keyboard shortcut">keyboard shortcuts</a>, typing indicator with customer message sneak-peek, <a href="/wiki/Location_awareness" title="Location awareness">geolocation</a>, <a href="/wiki/Tag_(metadata)" title="Tag (metadata)">tags system</a> for categorization, etc.</li></ul>\n<p>There are two ways of having an <a href="/wiki/Online_chat" title="Online chat">online chat</a> session between the agent and website visitor:\n</p>\n<ul><li>Passive communication, started when the website visitor clicks on the chat button or widget displayed on the page.</li>\n<li>Active communication, initiated manually by the agent or automatically by the LiveChat system as soon as the visitor meets the predefined criteria (i.e. searched keyword, time on website, encountered error, etc.).<sup id="cite_ref-Lynn_8-0" class="reference"><a href="#cite_note-Lynn-8">&#91;8&#93;</a></sup> Once the criteria is met, LiveChat displays a chat invitation with a customized message.<sup id="cite_ref-9" class="reference"><a href="#cite_note-9">&#91;9&#93;</a></sup> During the chat session, agents close the sales, which results in increasing the overall conversion rate.<sup id="cite_ref-10" class="reference"><a href="#cite_note-10">&#91;10&#93;</a></sup></li></ul>\n<p>In areas unrelated directly to the product, LiveChat relies on 3rd party services. This includes Postmark for the delivery of transactional emails,<sup id="cite_ref-11" class="reference"><a href="#cite_note-11">&#91;11&#93;</a></sup> Recurly for subscription billing,<sup id="cite_ref-12" class="reference"><a href="#cite_note-12">&#91;12&#93;</a></sup> <a href="/wiki/Pingdom" title="Pingdom">Pingdom</a> for performance monitoring and uptime tracking<sup id="cite_ref-13" class="reference"><a href="#cite_note-13">&#91;13&#93;</a></sup> and Upscope for instant screen sharing with customers.<sup id="cite_ref-14" class="reference"><a href="#cite_note-14">&#91;14&#93;</a></sup>\n</p>\n<h2><span class="mw-headline" id="Features">Features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a core features include: real-time <a href="/wiki/Website_monitoring" title="Website monitoring">website traffic monitoring</a>, built-in <a href="/wiki/Issue_tracking_system" title="Issue tracking Thai, Turkish, Ukrainian, Vietnamese, Tamil.\n</p>\n<h2><span class="mw-headline" id="Customer_Service_Platform">Customer Service Platform</span><span class="mw-editsection"><span
