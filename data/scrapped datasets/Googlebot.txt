FAQ</a></td></tr></tbody></table>\n<p><b>Googlebot</b> is the <a href="/wiki/Web_crawler" title="Web crawler">web crawler</a> software used by <a href="/wiki/Google" title="Google">Google</a>, which collects documents from the <a href="/wiki/World_Wide_Web" title="World Wide Web">web</a> to build a searchable index for the <a href="/wiki/Google_Search" title="Google Search">Google Search</a> engine. This name is actually used to refer to two different types of web crawlers: a desktop crawler (to simulate desktop users) and a mobile crawler (to simulate a mobile user).<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup>\n</p>\n<div id="toc" class="toc" role="navigation" aria-labelledby="mw-toc-heading"><input type="checkbox" role="button" id="toctogglecheckbox" class="toctogglecheckbox" style="display:none" /><div class="toctitle" lang="en" dir="ltr"><h2 id="mw-toc-heading">Contents</h2><span class="toctogglespan"><label class="toctogglelabel" for="toctogglecheckbox"></label></span></div>\n<ul>\n<li class="toclevel-1 tocsection-1"><a href="#Behavior"><span class="tocnumber">1</span> <span class="toctext">Behavior</span></a></li>\n<li class="toclevel-1 tocsection-2"><a href="#Mediabot"><span class="tocnumber">2</span> <span class="toctext">Mediabot</span></a></li>\n<li class="toclevel-1 tocsection-3"><a href="#References"><span class="tocnumber">3</span> <span class="toctext">References</span></a></li>\n<li class="toclevel-1 tocsection-4"><a href="#External_links"><span class="tocnumber">4</span> <span class="toctext">External links</span></a></li>\n</ul>\n</div>\n\n<h2><span class="mw-headline" id="Behavior">Behavior</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Googlebot&amp;action=edit&amp;section=1" title="Edit section: Behavior">edit</a><span class="mw-editsection-bracket">]</span></span></h2>\n<p>A website will probably be crawled by both Googlebot Desktop and Googlebot Mobile. However Google announced that, starting from September 2020, all sites were switched to mobile-first indexing, meaning Google is crawling the web using a smartphone Googlebot.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup> The subtype of Googlebot can be identified by looking at the user agent string in the request. However, both crawler types obey the same product token (useent token) in robots.txt, and so a developer cannot selectively target either Googlebot mobile or Googlebot desktop using robots.txt.\n</p><p>If a <a href="/wiki/Webmaster" title="Webmaster">webmaster</a> wishes to restrict the information on their site available to a Googlebot, or another well-behaved <a href="/wiki/Web_spider" class="mw-redirect" title="Web spider">spider</a>, they can do so with the appropriate directives in a <a href="/wiki/Robots.txt" class="mw-redirect" title="Robots.txt">robots.txt</a> file,<sup id="cite_ref-tools_3-0" class="reference"><a href="#cite_note-tools-3">&#91;3&#93;</a></sup> or by adding the <a href="/wiki/Meta_element" title="Meta element">meta tag</a> <code>&lt;meta name="Googlebot" content="nofollow" /&gt;</code> to the web page.<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup> Googlebot requests to <a href="/wiki/Web_server" title="Web server">Web servers</a> are identifiable by a <a href="/wiki/User-agent" class="mw-redirect" title="User-agent">user-agent</a> string containing "Googlebot" and a host address containing "googlebot.com".<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">&#91;<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (October 2019)">citation needed</span></a></i>&#93;</sup>\n</p><p>Currently, Googlebot follows <a href="/wiki/HREF" class="mw-redirect" title="HREF">HREF</a> links and SRC links.<sup id="cite_ref-tools_3-1" class="reference"><a href="#cite_note-tools-3">&#91;3&#93;</a></sup>  There is increasing evidence Googlebot can execute JavaScript and parse content generated by <a href="/wiki/Ajax_(programming)" title="Ajax (programming)">Ajax</a> calls as well.<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">&#91;5&#93;</a></sup> There are many theories regarding how advanced Googlebot\'s ability is to process JavaScript, with opinions ranging from minimal ability derived from custom interpreters.<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">&#91;6&#93;</a></sup> Currently, Googlebot uses a web rendering service (WRS) that is based on Chromium rendering engine (version 74 as on 7 May 2019).<sup id="cite_ref-7" class="reference"><a href="#cite_note-7">&#91;7&#93;</a></sup> Googlebot discovers pages by harvesting all the links on every page it finds. It then follows these links to other web pages. New web pages must be linked to from other known pages on the web in order to be crawled and indexed or manually submitted by the webmaster.\n</p><p>A problem that webmasters with low-bandwidth <a href="/wiki/Web_hosting" class="mw-redirect" title="Web hosting">Web hosting</a> plans<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">&#91;<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (May 2019)">citation needed</span></a></i>&#93;</sup> have often noted with the Googlebot is that it takes up an enormous amount of bandwidth.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">&#91;<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (March 2011)">citation needed</span></a></i>&#93;</sup>  This can cause websites to exceed their bandwidth limit and be taken down temporarily. This is especially troublesome for <a href="/wiki/Web_mirror" class="mw-redirect" title="Web mirror">mirror</a> sites which host many <a href="/wiki/Gigabyte" title="Gigabyte">gigabytes</a> of data. Google provides "<a href="/wiki/Google_Search_Console" title="Google Search Console">Search Console</a>" that allow website owners to throttle the crawl rate.<sup id="cite_ref-8" class="reference"><a href="#cite_note-8">&#91;8&#93;</a></sup>\n</p><p>How often Googlebot will crawl a site depends on the crawl budget. Crawl budget is an estimation of how often a website is updated.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">&#91;<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (May 2018)">citation needed</span></a></i>&#93;</sup> Technically, Googlebot\'s development team (Crawling and Indexing team) uses several defined terms internally to takes over what "crawl budget" stands for.<sup id="cite_ref-9" class="reference"><a href="#cite_note-9">&#91;9&#93;</a></sup> Since May 2019, Googlebot uses the latest <a href="/wiki/Chromium_(web_browser)" title="Chromium (web browser)">Chromium</a> rendering engine, which supports <a href="/wiki/ECMAScript_6" class="mw-redirect" title="ECMAScript 6">ECMAScript 6</a> features. This will make the bot a bit more "evergreen" and ensure that it is not relying on an outdated rendering engine compared to browser capabilities.<sup id="cite_ref-10" class="reference"><a href="#cite_note-10">&#91;10&#93;</a></sup>\n</p>\n<h2><span class="mw-headline" id="Mediabot">Mediabot</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Googlebot&amp;action=edit&amp;section=2" title="Edit section: Mediabot">edit</a><span class="mw-editsection-bracket">]</span></span></h2>\n<p><i>Mediabot</i> is the <a href="/wiki/Web_crawler" title="Web crawler">web crawler</a> that <a href="/wiki/Google" title="Google">Google</a> uses for analysing the content so <a href="/wiki/AdSense" class="mw-redirect" title="AdSense">Google AdSense</a> can serve <a href="/wiki/Contextual_advertising" title="Contextual advertising">contextually relevant</a> advertising to a web page. Mediabot identifies itself with the <a href="/wiki/User_agent" title="User agent">user agent</a> string "Mediapartners-Google/2.1".\n</p><p>Unlike other crawlers, Media bot does not follow links to discover new crawlable URLs, instead only visiting URLs that have included the AdSense code.<sup id="cite_ref-11" class="reference"><a href="#cite_note-11">&#91;11&#93;</a></sup> Where that content resides behind a login, the crawler can be given a log in so that it is able to crawl protected content.<sup id="cite_ref-12" class="reference"><a href="#cite_note-12">&#91;12&#93;</a></sup>\n</p>\n<h2><span class="mw-headline" id="References">References</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a
