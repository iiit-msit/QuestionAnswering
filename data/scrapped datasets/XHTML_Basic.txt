title="Comparison of layout engines (XHTML 1.1)">1.1</a></li></ul></li></ul></td>\n</tr></tbody></table><p><b>XHTML Basic</b> is an <a href="/wiki/XML" title="XML">XML</a>-based structured <a href="/wiki/Markup_language" title="Markup language">markup language</a> primarily used for simple (mainly <i><a href="/wiki/Handheld" class="mw-redirect" title="Handheld">handheld</a></i>) <a href="/wiki/User_agent" title="User agent">user agents</a>, typically mobile devices.\n</p><p>XHTML Basic is a subset of <a href="/wiki/XHTML_1.1" class="mw-redirect" title="XHTML 1.1">XHTML 1.1</a>, defined using <a href="/wiki/XHTML_Modularization" title="XHTML Modularization">XHTML Modularization</a> including a reduced set of modules for document structure, images, forms, basic tables, and object support. XHTML Basic is suitable for <a href="/wiki/Mobile_phone" title="Mobile phone">mobile phones</a>, <a href="/wiki/Personal_Digital_Assistant" class="mw-redirect" title="Personal Digital Assistant">PDAs</a>, <a href="/wiki/Pager" title="Pager">pagers</a>, and <a href="/wiki/Interactive_television" title="Interactive television">settop boxes</a>.\n</p><p>It will replace <a href="/wiki/Wireless_Markup_Language" title="Wireless Markup Language">WML</a> and <a href="/wiki/C-HTML" class="mw-redirect" title="C-HTML">C-HTML</a> as more compliant user agents are developed.\n</p><p>One large advantage XHTML Basic has over WML and C-HTML is that XHTML Basic pages can be rendered differently in <a href="/wiki/Web_browser" title="Web browser">web browsers</a> and on handhelds, without the need for two different versions of the same page.\n</p><p>In 2006, the specification was revised to version 1.1. Six new features have been incorporated into the language in order to better serve the small-device community.\n</p>\n<div id="toc" class="toc" role="navigation"
