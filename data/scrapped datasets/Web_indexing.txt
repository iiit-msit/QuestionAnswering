message</a>)</i></small></div></td></tr></tbody></table>\n<p><b>Web indexing</b>, or <b>internet indexing</b>, comprises methods for indexing the contents of a <a href="/wiki/Website" title="Website">website</a> or of the <a href="/wiki/Internet" title="Internet">Internet</a> as a whole. Individual websites or <a href="/wiki/Intranet" title="Intranet">intranets</a> may use a <a href="/wiki/Back-of-the-book_index" class="mw-redirect" title="Back-of-the-book index">back-of-the-book index</a>, while <a href="/wiki/Search_engines" class="mw-redirect" title="Search engines">search engines</a> usually use keywords and <a href="/wiki/Metadata_(computing)" class="mw-redirect" title="Metadata (computing)">metadata</a> to provide a more useful vocabulary for Internet or onsite searching. With the increase in the number of <a href="/wiki/Periodical" class="mw-redirect" title="Periodical">periodicals</a> that have articles online, web indexing is also becoming important for periodical websites.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup>\n</p><p>Back-of-the-book-style web indexes may be called "web site A-Z indexes".<sup id="cite_ref-kundu_2-0" class="reference"><a href="#cite_note-kundu-2">&#91;2&#93;</a></sup> The implication with "A-Z" is that there is an alphabetical browse view or interface. This interface differs from that of a browse through layers of hierarchical categories (also known as a <a href="/wiki/Taxonomy_(general)" class="mw-redirect" title="Taxonomy (general)">taxonomy</a>) which are not necessarily alphabetical, but are also found on some web sites. Although an A-Z index could be used to index multiple sites, rather than the multiple pages of a single site, this is unusual.\n</p><p><a href="/wiki/Metadata" title="Metadata">Metadata</a> web indexing involves assigning keywords, description or phrases to web pages or web sites within a <a href="/wiki/Metadata_tag" class="mw-redirect" title="Metadata tag">metadata tag</a> (or "meta-tag") field, so that the web page or web site can be retrieved with a list. This method is commonly used by <a href="/wiki/Search_engine_indexing" title="Search engine indexing">search engine indexing</a>.<sup id="cite_ref-3" class="reference"><a
