dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><div class="shortdescription nomobile noexcerpt noprint searchaux" style="display:none">Computer software</div>\n<p>An <b>offline reader</b> (sometimes called an <b>offline browser</b> or <b>offline navigator</b>) is <a href="/wiki/Computer_software" class="mw-redirect" title="Computer software">computer software</a> that downloads <a href="/wiki/E-mail" class="mw-redirect" title="E-mail">e-mail</a>, <a href="/wiki/Newsgroup" class="mw-redirect" title="Newsgroup">newsgroup</a> posts or <a href="/wiki/Web_page" title="Web page">web pages</a>, making them available when the computer is <a href="/wiki/Online_and_offline" title="Online and offline">offline</a>: not connected to the <a href="/wiki/Internet" title="Internet">Internet</a>.  Offline readers are useful for <a href="/wiki/Portable_computer" title="Portable computer">portable computers</a> and <a href="/wiki/Dial-up_access" class="mw-redirect" title="Dial-up access">dial-up access</a>.\n</p>\n<div id="toc" class="toc" role="navigation" aria-labelledby="mw-toc-heading"><input type="checkbox" role="button" id="toctogglecheckbox" class="toctogglecheckbox" section: Variations">edit</a><span class="mw-editsection-bracket">]</span></span></h2>\n<h3><span class="mw-headline" id="Website-mirroring_software">Website-mirroring software</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Offline_reader&amp;action=edit&amp;section=2" title="Edit section: Website-mirroring software">edit</a><span class="mw-editsection-bracket">]</span></span></h3>\n<p><b>Website mirroring software</b> is software that allows for the download of a copy of an entire <a href="/wiki/Website" title="Website">website</a> to the local hard disk for <a href="/wiki/Offline_browsing" class="mw-redirect" title="Offline browsing">offline browsing</a>. In effect, the downloaded copy serves as a <a href="/wiki/Mirror_website" class="mw-redirect" title="Mirror website">mirror</a> of the original site. <a href="/wiki/Web_crawler" title="Web crawler">Web crawler</a> software such as <a href="/wiki/Wget" title="Wget">Wget</a> can be used to generate a site mirror.\n</p>\n<h3><span class="mw-headline" id="Offline_mail_and_news_readers">Offline mail and news readers</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Offline_reader&amp;action=edit&amp;section=3" title="Edit section: Offline mail and news readers">edit</a><span class="mw-editsection-bracket">]</span></span></h3>\n<div role="note" class="hatnote navigation-not-searchable">See also: <a href="/wiki/Email_client" title="Email client">Email client</a> and <a href="/wiki/Newsreader_(Usenet)" title="Newsreader (Usenet)">Newsreader (Usenet)</a></div>\n<p><b>Offline mail readers</b> are <a href="/wiki/Computer_program" title="Computer program">computer programs</a> that allow users to read <a href="/wiki/Electronic_mail" class="mw-redirect" title="Electronic mail">electronic mail</a> or other messages (for example, those on <a href="/wiki/Bulletin_board_systems" class="mw-redirect" title="Bulletin board systems">bulletin board systems</a>) with a minimum of connection time to the server storing the messages. This is accomplished by the <a href="/wiki/Server_(computing)" title="Server (computing)">server</a> packaging up multiple messages into a <a href="/wiki/Data_compression" title="Data compression">compressed</a> file for the user to download and then disconnect. The user then reads the message packet locally and any replies or new messages generated are packaged up and uploaded back to the server upon the next connection. <b>Offline news readers</b> are similar, but the messages are organized into news groups.\n</p><p>Most e-mail protocols, like the common <a href="/wiki/Post_Office_Protocol" title="Post Office Protocol">POP</a>3 and <a href="/wiki/IMAP" class="mw-redirect" title="IMAP">IMAP</a>4 used for <a href="/wiki/Internet" title="Internet">internet</a> mail, need be on-line only during message transfer; the same applies to the <a href="/wiki/NNTP" class="mw-redirect" title="NNTP">NNTP</a> protocol used by <a href="/wiki/Usenet" title="Usenet">Usenet</a> (Network news).  Most end-user mailers, such as <a href="/wiki/Outlook_Express" title="Outlook Express">Outlook Express</a> and <a href="/wiki/AOL" title="AOL">AOL</a>, can be used offline even if they are mainly intended to be used online, but some mailers such as <a href="/wiki/Juno_Online_Services" title="Juno Online Services">Juno</a> are mainly intended to be used offline.\n</p><p>Off-line mail readers are generally considered to be those systems that did not originally offer such functionality, notably on <a href="/wiki/Bulletin_board_system" title="Bulletin board system">bulletin board systems</a> where toll charges and tying up telephone lines were a major concern. Users of large networks such as <a href="/wiki/FidoNet" title="FidoNet">FidoNet</a> regularly used offline mail readers, and it was also used for <a href="/wiki/UseNet" class="mw-redirect" title="UseNet">UseNet</a> messages on the internet, which is also an online system. The two most common formats for FidoNet BBS\'s were <a href="/wiki/Blue_Wave_(mail_reader)" title="Blue Wave (mail reader)">Blue Wave</a> and <a href="/wiki/QWK_(file_format)" title="QWK (file format)">QWK</a>. Less well-known examples include Silver Xpress\'s <a href="/wiki/FidoNet" title="FidoNet">OPX</a>, <a href="/wiki/XRS" class="mw-redirect" title="XRS">XRS</a>, <a href="/w/index.php?title=OMEM&amp;action=edit&amp;redlink=1" class="new" title="OMEM (page does not exist)">OMEM</a>, <a href="/w/index.php?title=SOUP_(file_format)&amp;action=edit&amp;redlink=1" class="new" title="SOUP (file format) (page does not exist)">SOUP</a> and <a href="/w/index.php?title=ZipMail&amp;action=edit&amp;redlink=1" class="new" title="ZipMail (page does not exist)">ZipMail</a>.\n</p>\n<h2><span class="mw-headline" id="List">List</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a
