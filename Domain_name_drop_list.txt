id="firstHeading" class="firstHeading" >Domain name drop list</h1>\n\t<div id="bodyContent" class="mw-body-content">\n\t\t<div id="siteSub" class="noprint">From Wikipedia, the free encyclopedia</div>\n\t\t<div id="contentSub"></div>\n\t\t<div id="contentSub2"></div>\n\t\t\n\t\t<div id="jump-to-nav"></div>\n\t\t<a class="mw-jump-link" href="#mw-head">Jump to navigation</a>\n\t\t<a class="mw-jump-link" href="#searchInput">Jump to search</a>\n\t\t<div id="mw-content-text" lang="en" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><p>A <b>domain name drop list</b> is a list containing the expired <a href="/wiki/Domain_names" class="mw-redirect" title="Domain names">domain names</a> that could be deleted from the <a href="/wiki/Domain_name_registry" title="Domain name registry">domain name registry</a> in the near future.  These lists are typically used by <a href="/wiki/Domainers" class="mw-redirect" title="Domainers">domainers</a> to locate expiring domain names with value.\n</p>\n<div id="toc" class="toc" role="navigation" aria-labelledby="mw-toc-heading"><input type="checkbox" role="button" id="toctogglecheckbox" class="toctogglecheckbox" style="display:none" /><div class="toctitle" lang="en" dir="ltr"><h2 id="mw-toc-heading">Contents</h2><span class="toctogglespan"><label class="toctogglelabel" for="toctogglecheckbox"></label></span></div>\n<ul>\n<li class="toclevel-1 tocsection-1"><a href="#Drop_list_contents"><span class="tocnumber">1</span> <span class="toctext">Drop list contents</span></a></li>\n<li class="toclevel-1 tocsection-2"><a href="#Dropcatching"><span class="tocnumber">2</span> <span class="toctext">Dropcatching</span></a></li>\n<li class="toclevel-1 tocsection-3"><a href="#See_also"><span class="tocnumber">3</span> <span class="toctext">See also</span></a></li>\n<li class="toclevel-1 tocsection-4"><a href="#References"><span class="tocnumber">4</span> <span class="toctext">References</span></a></li>\n</ul>\n</div>\n\n<h2><span class="mw-headline" id="Drop_list_contents">Drop list contents</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Domain_name_drop_list&amp;action=edit&amp;section=1" title="Edit section: Drop list contents">edit</a><span class="mw-editsection-bracket">]</span></span></h2>\n<p>There is no defined date range for data contained with a drop list, as they can contain anywhere between 1 day\'s worth of expiring domain names, to 30 or more days worth.  Some drop lists contain only domain names that follow the domain name deletion process (where the domain name enters the REGISTRAR-HOLD status, followed by REDEMPTIONPERIOD, and PENDINGDELETE), some contain only pre-release domain names, and some contain both pre-release and regular domain names.\n</p><p>The data contained within a drop list can also vary, with some lists providing only basic information, such as the domain name and its expiry date.  Other drop lists provide more detailed statistics, including, among others, <a href="/wiki/PageRank" title="PageRank">PageRank</a>, <a href="/wiki/Link_popularity" class="mw-redirect" title="Link popularity">Link popularity</a> and <a href="/wiki/Alexa_rank" class="mw-redirect" title="Alexa rank">Alexa rank</a>.\n</p><p>Critics say that promotion and marketing of so-called droplists may encourage cybercrime and payoffs at registrars, indicating that a registrants can lose their domains without abandon.\n</p>\n<h2><span class="mw-headline" id="Dropcatching">Dropcatching</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Domain_name_drop_list&amp;action=edit&amp;section=2" title="Edit section: Dropcatching">edit</a><span class="mw-editsection-bracket">]</span></span></h2>\n<div role="note" class="hatnote navigation-not-searchable">Main article: <a href="/wiki/Domain_drop_catching" title="Domain drop catching">Domain drop catching</a></div>\n<p>When a domain name is abandoned, or the period of validity is not renewed (expires), the webhost and registrar then action a "deletion" (the drop). Once a domain name has been deleted from the internet, it ceases to exist. Anyone may then register that name (the catch).  The process of re-registering expired names is known as <b>dropcatching</b> and various <a href="/wiki/Domain_name_registry" title="Domain name registry">domain name registries</a> have differing views on it.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup> Sometimes, people get locked out of their email and cannot reply to the renew request (or otherwise obstructed or hacked), and their domainname may be deleted and offered as available.\n</p>\n<h2><span class="mw-headline" id="See_also">See also</span><span class="mw-editsection"><span
